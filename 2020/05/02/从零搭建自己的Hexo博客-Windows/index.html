从零搭建自己的Hexo博客(Windows)

安装node.js和npm(廖雪峰老师)

https://www.liaoxuefeng.com/wiki/1022910821149312/1023025597810528

使用淘宝的镜像 npm install -g cnpm –registry=https://registry.npm.taobao.org

全局安装Hexo

cnpm install -g hexo-cli

hexo -v 验证一下

本地安装git（廖雪峰老师）

https://www.liaoxuefeng.com/wiki/896043488029600/896067074338496

在新建的博客地址文件夹中打开Git Bash Here,并输入hexo init 克隆

启动项目验证下是否克隆成功 hexo s 、hexo server 、npm run server 都可

创建一篇博客 hexo n “博客名”

清理一下 hexo clean（建议：在做一些修改之后都clean一下，避免一些未知错误）

博客生成 hexo g

启动 hexo s 运行成功后，在本地浏览器打开即可看到效果

github创建一个仓库，命名需要注意，要和自己GitHub的名字一致

在当前博客目录下下载一个插件 cnpm insatll hexo-deployer-git –save

到当前文件_config.yml中修改，如下：

默认在文件最底部

hexo clean (预防出现未知错误，真的是因为它出现好几个坑) 然后 hexo d（上传到GitHub创建的库）

如果出现下面这个错误

配置一下身份

git config –global user.name “用户的名字”
git config –global user.email“用户的邮箱”

上传成功后，就可以查询啦 地址：https://同户名.github.io
